## 📦 Installation

### Method 1: Upload Theme Files

1. Download the theme files
2. Go to your WordPress admin dashboard
3. Navigate to **Appearance > Themes**
4. Click **Add New > Upload Theme**
5. Choose the theme zip file and click **Install Now**
6. Click **Activate** to enable the theme

### Method 2: FTP Upload

1. Extract the theme files
2. Upload the `modern-landing-pro` folder to `/wp-content/themes/`
3. Go to **Appearance > Themes** in your WordPress admin
4. Find "Modern Landing Pro" and click **Activate**

## 🔧 Development

### File Structure

```
modern-landing-pro/
├── style.css              # Main stylesheet with theme info
├── index.php              # Main template file
├── functions.php          # Theme functions and features
├── screenshot.png         # Theme screenshot
├── README.md             # Documentation

└── assets/
    ├── images/           # Theme images
    └── css/             # Additional stylesheets
```

### Customizing the Theme

#### Adding New Sections

1. Open `index.php`
2. Add your section HTML before the footer
3. Add corresponding CSS in `style.css`
4. Add any JavaScript functionality in `js/theme.js`

#### Modifying Colors

1. Open `style.css`
2. Find the CSS custom properties or color values
3. Replace with your desired colors
4. Update gradient definitions as needed

#### Adding Custom Fonts

1. Enqueue your fonts in `functions.php`
2. Update the font-family in `style.css`
3. Preload fonts for better performance
